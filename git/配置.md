# 配置 username,email

git config --global user.name 'yourname'
git config --global user.email 'youremail'

## config 作用域

--local 只对某个仓库有效(如果 local 和 global 设置了不同的用户名,优先选择 local 设置的)
--global 对当前用户的所有仓库都有效 (常用)
-- system 对系统所有登录的用户都有效

## 查看配置信息

git config --list --global
